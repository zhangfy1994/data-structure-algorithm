## 二叉树

- 二叉树每个节点最多有两个子节点

<img src="../../img/binary-tree.jpeg">

```js
// 二叉树结构
const binaryTree = {
  val: "a",
  left: {
    val: "b",
    left: {
      val: "c",
    },
    right: {
      val: "d",
    },
  },
  right: {
    val: "e",
    left: {
      val: "f",
    },
    right: {
      val: "g",
    },
  },
};
```

- 二叉树深度遍历分为 先序遍历，中序遍历，后序遍历
